# **Shell Script**

## ✅개념
**Shell Script**는 <br>
리눅스/맥OS 터미널에서 실행되는 명령어 묶음 자동화 파일이다. <br>
<br>
즉, <br>
> 사람이 터미널에서 치는 명렁어를 파일로 저장해서 자동 실행하는 것

## 왜 쓰는것인가?
- 서버 배포 자동화
- 빌드 자동화
- 테스트 자동화
- Docker 실행
- 로그 정리
- 반복 작업 자동화

백엔드에서 **CI/CD의 기본 도구** <br>
<br>
## 기본 구조
```
#!/bin/bash

echo "Hello World"
```
| 문법  | 의미 |
|-----|----|
| #!/bin/bash | 이 파일을 bash로 실행하겠다는 선언 |
| echo  | 출력 |
| $변수명  | 변수 사용 |
