📦 Poetry - 의존성 관리 도구 <br>
<br>
💡 개요 <br>
Poetry는 Python 프로젝트의 의존성 관리와 패키지 빌드를 담당하는 현대적인 도구입니다. <br>
<br>
⭐ 주요 기능 
<br>

📚 의존성 관리 - pyproject.toml에서 프로젝트 의존성을 선언적으로 관리 <br>
🔒 버전 잠금 - poetry.lock 파일로 정확한 패키지 버전 고정 <br>
🌍 가상환경 자동 관리 - 프로젝트별 격리된 환경 자동 생성 <br>
📦 패키지 빌드 및 배포 - PyPI에 쉽게 배포 가능 <br>
<br>
```
┌─────────────────────────────────────────────────┐
│              pyproject.toml                     
│  (프로젝트 설정 & 의존성 선언)                     
└────────────────┬────────────────────────────────┘
                 │
                 ▼
         poetry install
                 │
                 ▼
┌─────────────────────────────────────────────────┐
│              poetry.lock                        
│  (정확한 버전 잠금 파일)                           
└────────────────┬────────────────────────────────┘
                 │
                 ▼
┌─────────────────────────────────────────────────┐
│            가상환경 (.venv)                      
│  패키지들이 설치된 격리된 환경                      
└─────────────────────────────────────────────────┘
```

🎨 Black - 코드 포맷터 <br>
<br>
💡 개요 <br>
Black은 타협 없는(uncompromising) Python 코드 포맷터입니다. <br>
"코드 스타일을 고민할 시간에 코드를 작성하자"는 철학을 가지고 있습니다. <br>
<br>
⭐ 주요 기능
<br>

🔧 자동 포맷팅 - 일관된 코드 스타일 자동 적용 <br>
⚫ 단일 스타일 - 설정 옵션이 최소화되어 논쟁 종료 <br>
🚀 빠른 속도 - Rust로 재작성된 고성능 포맷터 <br>
🔄 결정론적 - 동일한 입력에 항상 동일한 출력 <br>

🔄 작동 흐름
<br>
```
┌─────────────┐
│  소스 코드   
│   (.py)     
└──────┬──────┘
       │
       ▼
┌─────────────────────┐
│   Black 포맷터       
│  • 들여쓰기 수정     
│  • 공백 정리        
│  • 줄 길이 조정     
│  • 따옴표 통일      
└──────┬──────────────┘
       │
       ▼
┌─────────────┐
│ 포맷팅된     
│  코드 출력   
└─────────────┘
```

🔍 Ruff - 초고속 린터
<br>
💡 개요 <br>
Ruff는 Rust로 작성된 초고속 Python 린터입니다. <br>
Flake8, isort, pylint 등 여러 도구를 하나로 통합했습니다. <br>
<br>
⭐ 주요 기능
<br>

⚡ 극도로 빠름 - 기존 린터 대비 10-100배 빠른 속도 <br>
🔄 자동 수정 - 많은 오류를 자동으로 고칠 수 있음 <br>
📦 올인원 - 여러 린팅 도구의 기능을 하나로 통합 <br>
🎯 700+ 규칙 - 다양한 코딩 규칙 검사 <br>

<br>
🔍 검사 항목
<br>

```
┌─────────────────────────────────────────────────┐
│               Ruff 검사 항목                      
├─────────────────────────────────────────────────┤
│  ✓ 코드 품질 (pyflakes)                          
│    - 사용하지 않는 import                         
│    - 정의되지 않은 변수                           
│    - 사용되지 않는 변수                           
│                                                 
│  ✓ 코드 스타일 (pycodestyle)                     
│    - PEP 8 준수 여부                             
│    - 줄 길이, 들여쓰기                            
│                                                 
│  ✓ Import 정리 (isort)                           
│    - import 문 자동 정렬                          
│    - 그룹별 구분                                  
│                                                 
│  ✓ 복잡도 검사 (mccabe)                          
│    - 함수 복잡도 측정                             
│                                                 
│  ✓ 보안 취약점 (bandit)                          
│    - 잠재적 보안 문제 감지                         
└─────────────────────────────────────────────────┘
```

🔒 Mypy - 정적 타입 검사기
<br>

💡 개요 <br>

Mypy는 Python의 정적 타입 검사 도구입니다. <br>
코드 실행 전에 타입 오류를 미리 발견할 수 있습니다. <br>
<br>
⭐ 주요 기능
<br>

🎯 타입 오류 사전 감지 - 런타임 전에 타입 문제 발견 <br>
📝 점진적 타입 검사 - 필요한 부분만 타입 힌트 추가 가능 <br>
🔧 IDE 통합 - 실시간 타입 검사 지원 <br>
📚 표준 라이브러리 지원 - Python 표준 라이브러리 타입 정의 내장 <br>

<br>


```
┌─────────────────────┐
│   타입 힌트가 있는     
│    Python 코드       
│                     
│  def add(a: int,    
│          b: int)   
│       -> int:      
│    return a + b     
└──────────┬──────────┘
           │
           ▼
    ┌──────────────┐
    │    Mypy      
    │  타입 검사기   
    └──────┬───────┘
           │
           ▼
    ┌─────────────────────┐
    │  타입 오류 리포트      
    │                     
    │  ✓ 타입 일치         
    │  ✗ 타입 불일치        
    │  ✗ None 체크 누락     
    └─────────────────────┘
```
