🐍 Python 개발 도구 가이드
<br>
이 프로젝트는 Python 개발의 필수 도구들을 활용합니다.
각 도구의 역할과 사용법을 알아보세요.
<br>
<br>

<br>
📦 Poetry - 의존성 관리 도구
<br>
💡 개요
Poetry는 Python 프로젝트의 의존성 관리와 패키지 빌드를 담당하는 현대적인 도구입니다.
<br>
⭐ 주요 기능
<br>

📚 의존성 관리 - pyproject.toml에서 프로젝트 의존성을 선언적으로 관리
🔒 버전 잠금 - poetry.lock 파일로 정확한 패키지 버전 고정
🌍 가상환경 자동 관리 - 프로젝트별 격리된 환경 자동 생성
📦 패키지 빌드 및 배포 - PyPI에 쉽게 배포 가능

<br>
🔄 작동 원리
<br>
```
┌─────────────────────────────────────────────────┐
│              pyproject.toml                     │
│  (프로젝트 설정 & 의존성 선언)                     │
└────────────────┬────────────────────────────────┘
                 │
                 ▼
         poetry install
                 │
                 ▼
┌─────────────────────────────────────────────────┐
│              poetry.lock                        │
│  (정확한 버전 잠금 파일)                           │
└────────────────┬────────────────────────────────┘
                 │
                 ▼
┌─────────────────────────────────────────────────┐
│            가상환경 (.venv)                      │
│  패키지들이 설치된 격리된 환경                      │
└─────────────────────────────────────────────────┘
```
<br>
🎨 Black - 코드 포맷터
<br>
💡 개요
Black은 타협 없는(uncompromising) Python 코드 포맷터입니다.
"코드 스타일을 고민할 시간에 코드를 작성하자"는 철학을 가지고 있습니다.
<br>
⭐ 주요 기능
<br>

🔧 자동 포맷팅 - 일관된 코드 스타일 자동 적용
⚫ 단일 스타일 - 설정 옵션이 최소화되어 논쟁 종료
🚀 빠른 속도 - Rust로 재작성된 고성능 포맷터
🔄 결정론적 - 동일한 입력에 항상 동일한 출력

🔄 작동 흐름
<br>
```
┌─────────────┐
│  소스 코드   │
│   (.py)     │
└──────┬──────┘
       │
       ▼
┌─────────────────────┐
│   Black 포맷터       │
│  • 들여쓰기 수정     │
│  • 공백 정리        │
│  • 줄 길이 조정     │
│  • 따옴표 통일      │
└──────┬──────────────┘
       │
       ▼
┌─────────────┐
│ 포맷팅된     │
│  코드 출력   │
└─────────────┘
```
